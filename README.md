# Redspher Calculator
Test project for Redspher.

## `Front: React v17 with typescript`
### How to run
```
cd front/
npm install
npm start
npm test // to test
```
## `Back: Express with typescript`
### How to run
```
cd back/
npm install
npm run dev
npm test // to test
```

## With Docker (dev)
Build images docker compose
(this command builds front & back images and starts them)
```
docker compose up
```
